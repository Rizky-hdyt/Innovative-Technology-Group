# Project Brief: Sistem Perpustakaan Buku

## Tujuan

Membangun sistem perpustakaan berbasis web yang membantu pengelolaan buku, peminjaman, dan pengembalian secara digital sehingga proses administrasi menjadi lebih cepat, tertata, dan mudah dipantau.

## Ruang Lingkup

Sistem mencakup pengelolaan data buku oleh admin, pencarian buku oleh anggota, transaksi peminjaman buku, proses pengembalian buku, serta perhitungan denda keterlambatan secara otomatis.

## Pengguna

### Admin/Petugas Perpustakaan

- Mengelola data buku.
- Melihat data peminjaman.
- Memverifikasi pengembalian.
- Memantau denda keterlambatan.

### Anggota/Mahasiswa

- Mencari buku.
- Melihat stok buku.
- Meminjam buku yang tersedia.
- Melihat riwayat peminjaman.

## Fitur Utama

- Manajemen data buku.
- Peminjaman buku.
- Pengembalian buku.
- Perhitungan denda keterlambatan.
- Riwayat peminjaman anggota.

## Batasan Awal

- Sistem berbasis web.
- Admin memiliki akses penuh ke pengelolaan data buku.
- Anggota hanya dapat meminjam buku jika stok tersedia.
- Denda dihitung otomatis oleh sistem.
